新增類別 系統使用紀錄
Repository.ESK.Domain.UseLog

在類別Repository.ESK.ESKContext
新增屬性
public DbSet<UseLog> UseLogs { get; set; }

開啟檢視 => 其他視窗 => 套件管理器主控台
